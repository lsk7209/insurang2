# 수동 수정 가이드

## 발견된 오류 3가지

### 오류 1: 1121번째 줄 - HTML 엔티티 문제

**현재 코드:**
```tsx
JSON 배열 형식으로 입력하세요. 예: [&#123;&quot;title&quot;: &quot;제목&quot;, &quot;text&quot;: &quot;내용&quot;&#125;]
```

**수정 방법:**
1. `app/admin/offers/page.tsx` 파일을 엽니다
2. 1121번째 줄을 찾습니다
3. 다음 중 하나로 변경합니다:

**옵션 A: HTML 엔티티를 일반 문자로 변경**
```tsx
JSON 배열 형식으로 입력하세요. 예: [{"title": "제목", "text": "내용"}]
```

**옵션 B: JSX 표현식 사용**
```tsx
JSON 배열 형식으로 입력하세요. 예: [&#123;&quot;title&quot;: &quot;제목&quot;, &quot;text&quot;: &quot;내용&quot;&#125;]
```

**옵션 C: 예시 텍스트 간소화 (권장)**
```tsx
JSON 배열 형식으로 입력하세요. 예: [{"title": "제목", "text": "내용"}]
```

### 오류 2: 593번째 줄 - 닫히지 않은 div 태그

**확인 사항:**
- 593번째 줄: `<div className="p-6">` 시작
- 1183번째 줄: `</div>` 닫힘 확인 필요

**수정 방법:**
1. 593번째 줄부터 1183번째 줄까지의 div 태그 구조를 확인합니다
2. 모든 `<div>` 태그가 올바르게 닫혀있는지 확인합니다
3. 닫히지 않은 태그가 있다면 `</div>`를 추가합니다

### 오류 3: 1184번째 줄 - 괄호 문제

**확인 사항:**
- 1184번째 줄: `</form>` 앞의 괄호 구조 확인
- 1126번째 줄의 `)}`와 연결 확인

**수정 방법:**
1. 1126번째 줄의 `)}`가 올바른 위치에 있는지 확인
2. 1184번째 줄의 `</form>` 앞에 필요한 괄호가 있는지 확인

## 수정 후 확인

1. 파일 저장
2. 터미널에서 다음 명령어 실행:
   ```bash
   npm run build
   ```
3. 빌드 성공 확인
